# Quotes API Project

## Features
- Fetches random quotes from API Ninjas
- Displays 1 quotes per request per day
- Handles errors (network, API, authentication, timeout)
- User-friendly UI 

## Usage
1. Open `quotes.html` in a browser.
2. Get Quotes to load on your screen or TV.

## Deployment
Can be hosted on GitHub Pages or any static host.
